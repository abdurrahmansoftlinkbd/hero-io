# Hero.io

A modern, fully responsive app store web application built with React 19. Browse trending apps, view detailed app information, install your favorites, and manage your installed apps all with smooth animations and a polished UI.

---

## Live Demo

[View Live](https://hero-io-one.vercel.app/)

---

## Features

- **Trending Apps** Browse the top 8 trending apps on the homepage
- **Live Search** Filter apps by title in real time
- **App Details** View ratings chart, downloads, reviews, and description
- **Install Apps** Install apps and persist them to `localStorage`
- **Uninstall Apps** Remove apps from your installed list
- **Installation Page** View and manage all your installed apps
- **Sort by Downloads** Sort installed apps High→Low or Low→High
- **Fully Responsive** Optimized for mobile, tablet, and desktop
- **Dynamic Page Titles** Page title updates on every route change
- **Skeleton Loaders** DaisyUI skeletons while data is loading
- **Toast Notifications** Success and error feedback on every action
